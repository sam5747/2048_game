2048 Game rules

  • The game concept is simple and complex at the same time.
  • In this game, the player must combine tiles containing the same numbers until they reach the number 2048. The tiles can contain only integer values starting from 2, and that are a power of two, like 2, 4, 8, 16, 32, and so on.
  • To only rule of the game is that you have to merge 2 tiles with the same numerical value and then they become 1 tile with x2 of their original value then you have to merge this new x2 tile with another x2 tile until you reach 2048.
